# Workspace Scoping & Constraints

## Allowed Directory
* You are strictly restricted to modifying files within the `.carecloud-voice-agent` directory (Replace with your specific path).
* You may create, edit, or delete files and folders inside this directory as needed to fulfill tasks.

## System Changes & Packages
* Do NOT attempt to install or remove global npm packages without asking first.
* Do NOT attempt to install or remove Ubuntu (`apt`) packages without asking first.
* Always prompt the user for confirmation before executing any command that modifies the global system environment outside the project workspace.
